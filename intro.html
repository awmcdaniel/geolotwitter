GeoloTweets: Introduction

Welcome to a (relatively) short tutorial on the HTML5 geolocation API, the twitter API, and the twitteroauth library. We'll learn about all three elements in the course of creating a web interface that will get a user's geographical location data, send that data to twitter, and display for the user a list of tweets from nearby.

Try my implementation of Geolotwitter.
