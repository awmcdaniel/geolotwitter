Obtaining twitter API information

First, you’ll need a twitter account. If you don’t have one, signing up is simple. Once you have an account, visit https://apps.twitter.com/ and sign in. You’ll be presented with a screen indicating that you have no apps (unless you do! then they’ll be listed here). Click on Create New App, and fill in the name of your app, description, and agree to the developer agreement.

Once your app is set up, click on the Keys and Access Tokens tab, and then at the bottom of the page, click Create My Access Token. Once created, this page will display 4 bits of information you’ll need to access the twitter API: the Consumer Key (API Key), the Consumer Secret (API Secret), the Access Token, and the Access Token Secret.

Now we have the information we need to make a request to the twitter API. Kind of.
